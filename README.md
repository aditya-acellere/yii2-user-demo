# Yii2-user demo

This is source code of Yii2-user demo.

## Installation

```bash
$ php composer.phar create-project --prefer-dist --stability=dev dektrium/yii2-user-demo yii2-user
$ cd yii2-user
$ php yii migrate/up --migrationPath=@vendor/dektrium/yii2-user/migrations
```
